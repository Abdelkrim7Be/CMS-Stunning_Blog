# CMS Stunning Blog

A modern, responsive Content Management System built with PHP, MySQL, and Bootstrap.

## Features

- **User Authentication** - Secure admin login system
- **Post Management** - Create, edit, and delete blog posts
- **Category System** - Organize posts by categories
- **Comment System** - Manage and approve user comments
- **Admin Dashboard** - Monitor blog statistics and activity
- **Responsive Design** - Works perfectly on all devices

## Installation

1. Clone the repository
```bash
git clone git@github.com:Abdelkrim7Be/CMS-Stunning_Blog.git
```

2. Import the database
```bash
mysql -u username -p database_name < C.M.S.sql
```

3. Configure database connection in `Includes/DB.php`

4. Access the application through your web server

## Pages

- **Blog.php** - Main blog page with posts listing
- **Dashboard.php** - Admin dashboard with statistics
- **Login.php** - Admin authentication
- **Posts.php** - Manage all posts
- **Categories.php** - Manage categories
- **Comments.php** - Moderate comments
- **Admins.php** - Manage admin users
